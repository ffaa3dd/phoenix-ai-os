# src/memory - Memory Systems

أنظمة الذاكرة المتعددة:
1. Working Memory - الذاكرة الفورية
2. Short Memory - الذاكرة قصيرة المدى (24h)
3. Long Memory - الذاكرة طويلة المدى
4. Trading Memory - ذاكرة التداولات
5. Prompt Memory - ذاكرة الـ Prompts
6. Semantic Memory - الذاكرة الدلالية

## البنية:
```
memory/
├── working_memory/     # الذاكرة الفورية
├── short_memory/       # 24 ساعة
├── long_memory/        # تاريخي
├── trading_memory/     # كل التداولات
├── prompt_memory/      # prompts فعالة
├── semantic_memory/    # معرفة السوق
└── query_engine/       # محرك البحث
```
