# src/mobile - Flutter Mobile Application

تطبيق الجوال متعدد المنصات

## الميزات:
- Dashboard بالبيانات الفورية
- Portfolio Management
- Risk Meter
- AI Chat Interface
- Trade History & Replay
- Push Notifications
- Settings

## البنية:
```
mobile/
├── lib/
│   ├── screens/        # الشاشات
│   ├── widgets/        # العناصر
│   ├── services/       # الخدمات
│   ├── models/         # النماذج
│   └── utils/          # الأدوات
├── test/              # الاختبارات
└── pubspec.yaml       # المتطلبات
```

## المتطلبات:
- Flutter 3.0+
- Dart 3.0+
