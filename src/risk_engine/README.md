# src/risk_engine - Risk Management Engine

محرك إدارة المخاطر الشامل

## العناصر:
- Risk Score Calculator - حساب درجة المخاطر
- Position Sizer - تحديد حجم المركز
- Correlation Analyzer - تحليل الارتباط
- Drawdown Monitor - مراقب الهبوط
- Daily Loss Limiter - حد الخسارة اليومية
- Circuit Breaker - قاطع الدوائر
- Emergency Stop - التوقف الطارئ

## البنية:
```
risk_engine/
├── calculators/        # أدوات الحساب
├── monitors/          # المراقبات
├── limiters/          # المحددات
├── policies/          # السياسات
└── utils/             # أدوات مساعدة
```
