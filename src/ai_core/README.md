# src/ai_core - AI Core System

هذا المجلد يحتوي على:
- 12 وكيل AI متخصص
- نظام الذاكرة المتعددة
- محرك القرارات
- نظام التعلم

## البنية:
```
ai_core/

├── memory/              # أنظمة الذاكرة
├── decision_engine/     # محرك القرارات
├── utils/              # أدوات مساعدة
├── config.py           # الإعدادات
├── requirements.txt    # المتطلبات
└── main.py            # نقطة الدخول
```
