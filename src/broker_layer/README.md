# src/broker_layer - Broker Layer

طبقة الوسيط - التحكم في تنفيذ الأوامر والسياسات

## المسؤوليات:
- تحقق من صحة الأوامر
- حساب المخاطر
- تطبيق السياسات
- تنفيذ الصفقات
- إدارة المراكز

## البنية:
```
broker_layer/
├── validators/         # التحقق من الصحة
├── risk_calculator/    # حاسب المخاطر
├── policy_engine/      # محرك السياسات
├── executor/           # محرك التنفيذ
└── utils/             # أدوات مساعدة
```
