# tests/ - Test Suite

مجموعة الاختبارات الشاملة

## أنواع الاختبارات:
- **unit/** - اختبارات الوحدات
- **integration/** - اختبارات التكامل
- **e2e/** - اختبارات النهاية للنهاية
- **risk/** - اختبارات سيناريوهات المخاطر
- **market_replay/** - اختبارات المراجعة السوقية
- **fixtures/** - بيانات الاختبار

## متطلبات:
- 85%+ coverage
- جميع الاختبارات يجب أن تمر
- اختبارات الأداء قبل الإطلاق
